O que é Engenharia de Prompt?
Engenharia de prompt é a forma de escrever pedidos (comandos) para uma inteligência artificial, como o ChatGPT, de um jeito que ela entenda exatamente o que você quer.

Pense assim:
👉 A IA é muito inteligente, mas não lê sua mente.
👉 Quanto mais claro e específico for seu pedido, melhor será a resposta.

📌 Explicando de forma simples

É como pedir um lanche:

❌ Pedido ruim:

 Me dá um sanduíche.

✅ Pedido bom:

Quero um sanduíche de frango grelhado, sem tomate, no pão integral e com pouco molho.

Quanto mais detalhes importantes você dá, melhor o resultado.

🧠 Como fazer um bom prompt?

Um bom prompt geralmente tem:

1. Objetivo claro – O que você quer?
2. Contexto – Para quem é? Qual situação?
3. Formato da resposta – Texto curto? Lista? Formal? Criativo?
4. Restrições (se necessário) – Tamanho, tom, público, etc.

🎯 Regra de ouro

Se a resposta vier ruim, o problema geralmente não é a IA…
É o prompt que pode ser melhorado.

Engenharia de prompt é basicamente aprender a:

Ser claro
Ser específico
Dar contexto
Dizer como você quer a resposta
